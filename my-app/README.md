# FullTech Catalog

Catálogo de produtos desenvolvido com React + Vite.

## Funcionalidades

- Listagem dinâmica de produtos
- Adição de novos produtos
- Filtro de busca
- Responsividade básica
- Loading screen

## Tecnologias

- React
- Vite
- CSS3
- JavaScript

## Como rodar

npm install
npm run dev
